Converts a number from string to numeric representation.

Parameters
[in]source_stringA string representation of a number.
[in]number_baseA base of a number.
Returns
A numeric representation of a number.
No-throw guarantee
Strong.
Exceptions
std::runtime_errorThrown if source_string does not represent a valid_number. Also thrown if source_string represents a number which is out of range of long int.

Converts a number represented by a source_string to a long int. Treats a source_string as a numeral with a base of number_base.
